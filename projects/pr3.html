NEXT WORD RECOMMENDER

JAVA

Word completion works when the writer writes the first word, the program predicts one or more possible words as choices. If the word he intends to write is included in the list he can select it. If the word that the user wants is not predicted, the writer must enter the next letter of the word. At this time, the word choice(s) is altered so that the words provided begin with the same letters as those that have been selected. When the word that the user wants appears it is selected, and the word is inserted into the text.
